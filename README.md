# 🌱 Raízes da Cultura

Projeto desenvolvido para o Agrinho 2026.

## Tema

Agro Forte, Futuro Sustentável: Equilíbrio entre Produção e Meio Ambiente

## Objetivo

Mostrar como a agricultura contribui para a cultura brasileira através das festas, tradições, gastronomia e práticas sustentáveis.
